# Настройка WhatsApp-рассылки в KURSOR

## Что добавлено

- **Автоматические напоминания**: ежедневно в заданное время бот проверяет расписание и пишет родителям в WhatsApp о завтрашнем (или любом другом) занятии.
- **Шаблоны сообщений**: несколько шаблонов с переменными `{{parentName}}`, `{{studentName}}`, `{{groupName}}`, `{{time}}`, `{{date}}`.
- **Нормализация номеров**: `87771234567` → `77771234567`, `+7…` — автоматически.
- **Тест-отправка**: из панели можно отправить тестовое сообщение на любой номер.
- **Ручной запуск**: запустить рассылку прямо сейчас без ожидания расписания.
- **Лог отправок**: история всех сообщений с статусами OK/Ошибка.
- **Подсказка при вводе номера**: в карточке клиента сразу видно нормализованный номер.

## Шаг 1: Зарегистрироваться на Green API

1. Перейдите на **https://green-api.com**
2. Создайте аккаунт (бесплатно)
3. Нажмите «Создать инстанс»
4. В интерфейсе инстанса нажмите **«Подключить телефон»** и отсканируйте QR-код **корпоративным WhatsApp** (телефон должен быть постоянно онлайн)
5. Скопируйте **ID инстанса** (число вида `1234567890`) и **API Token**

> **Бесплатный тариф**: 100 сообщений/сутки — достаточно для большинства школ.  
> **Платный Developer**: от $5/мес, неограниченно.

## Шаг 2: Настроить в админ-панели

1. Войдите в **Админ → CRM → WhatsApp рассылка**
2. Введите **Instance ID** и **API Token**
3. Выберите время отправки (например, 18:00)
4. Выберите «за сколько дней» (обычно 1 день)
5. Нажмите **«Сохранить настройки»**
6. Включите тумблер **«🟢 Рассылка включена»**
7. Нажмите **«Отправить тест»** — введите свой номер и проверьте

## Шаг 3: Убедиться что номера введены правильно

В **Админ → CRM → Клиенты** откройте карточку клиента.  
При вводе номера в поле «Телефон родителя» появится подсказка: `→ +77771234567 (готово для WhatsApp)`.

Поддерживаемые форматы:
- `+7 777 123 45 67`
- `87771234567`
- `77771234567`
- `7771234567` (10 цифр — добавляется 7)

## Шаг 4: Расписание групп

Убедитесь что у групп заполнено расписание (**Журнал → Группы → ⚙ Расписание**).  
Бот проверяет таблицу `group_schedule` и находит занятия завтрашнего дня недели.

## Переменные шаблона

| Переменная | Что подставляется |
|---|---|
| `{{parentName}}` | Имя родителя из карточки CRM |
| `{{studentName}}` | ФИО ученика |
| `{{groupName}}` | Название группы |
| `{{time}}` | Время занятия (из расписания) |
| `{{date}}` | Дата занятия (например, «15 января») |
| `{{weekday}}` | День недели («понедельник», «вторник»…) |

## Деплой на Railway / VPS

Переменные окружения устанавливаются в настройках Railway:
```
JWT_SECRET=...
PORT=3000
```
Настройки Green API хранятся в SQLite и задаются через UI — переменные окружения не нужны.
